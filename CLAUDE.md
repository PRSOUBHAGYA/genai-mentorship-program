# GenAI Mentorship Program Website Design

## Project Overview
A minimalist, high-conversion frontend website to promote the GenAI Mentorship Program and enroll students.

- **Goal:** Inform students about the program and capture enrollment details.
- **Hosting:** GitHub Pages (Static).
- **Duration:** 2 Months (Live Cohort).
- **Target Audience:** Open to everyone—students, professionals, and career-switchers from any industry or technical background.

## Design Guidelines
- **Aesthetic:** Minimalist, informative, and attention-grabbing.
- **Visual Style:** Minimalist, professional with an interactive "Connecting Dots" background pattern (dots repel on mouse hover).
- **UX Principle:** "Less is more." Clear hierarchy, plenty of whitespace, and a strong Call to Action (CTA).


## Website Structure (Sitemap)
1. **Hero Section:** 
   - Catchy headline.
   - Sub-headline emphasizing "Live Cohort" and "2 Months Duration".
   - Primary CTA: "Enroll Now".
2. **About the Program:** 
   - Brief explanation of the mentorship value.
   - Focus on the transition from ML to GenAI.
3. **Course Curriculum:** 
   - Organized sections based on the program roadmap.
   - **Modules:**
     - Intro to GenAI (LLM History, Prompt Engineering).
     - Learning Python with GenAI (Ollama, Data Types, OOP for Prompts).
     - Advanced GenAI Concepts (Langchain, RAG, Langgraph, Agents).
     - Capstone Projects (HR Policy Assistant, ATS Resume Builder).
     - Vibe Coding & Portfolio Building.
4. **Enrollment Form:**
   - Fields: Name, Email, Phone Number, Professional Status (Student/Professional), and other relevant details.
   - Data Destination: [TBD - Pending User Input]
5. **Footer:**
   - Social links (LinkedIn, YouTube).
   - Copyright and Contact info.

## Technical Constraints & Instructions
- **Frontend only:** HTML/CSS/JS or a framework like React/Tailwind.
- **Static Hosting:** Must be compatible with GitHub Pages.
- **Data Capture:** Since GitHub Pages is static, a third-party service or GitHub Action must be used to save student details to a CSV.
